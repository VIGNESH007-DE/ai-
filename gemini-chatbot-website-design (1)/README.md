# Gemini Flash AI Chatbot

A professional, simple, and attractive chatbot website using the Google Gemini 1.5 Flash model.

## 🚀 Getting Started

1.  **Get your Gemini API Key**: Visit [Google AI Studio](https://aistudio.google.com/) to get a free API key.
2.  **Add your key**: Create a `.env` file in the root of the project and add your key:
    ```env
    VITE_GEMINI_API_KEY=your_key_here
    ```
3.  **Run the app**:
    ```bash
    npm install
    npm run dev
    ```

## ✨ Features
- **Clean UI**: Minimalist design with dark/light mode.
- **Responsive**: Works great on mobile and desktop.
- **Smart**: Powered by Gemini 1.5 Flash for fast, intelligent responses.
- **Markdown**: Full support for code snippets, lists, and bold text.
